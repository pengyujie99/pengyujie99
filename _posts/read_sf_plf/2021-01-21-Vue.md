## Vue
<br>前端的渐进式框架,只做视图层
<br>
<br>原来的前端框架
<br>react：虚拟dom
<br>angular：mvvm架构
<br>
<br>vue则相当于是这二个的集大成
<br>vue常用七大属性：
<br>el属性
<br>data属性
<br>template属性
<br>methods属性
<br>render属性
<br>computed属性
<br>watch属性

## 使用方法时，需要进行绑定，要用到v-on，导入xmlns:v-on="http://www.w3.org/1999/xhtml"
```html
<!DOCTYPE html>
<html lang="en" xmlns:v-on="http://www.w3.org/1999/xhtml">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script src="./vue.min.js"></script>
</head>
<body>
<div id="app">
    <button v-on:click="sayHi"> click me </button>
    {{message}}
</div>


<script>
    var vm = new Vue({
        el: "#app",
        data: {
            message: "hello!"
        },
        methods:{
            sayHi: function (event){
                alert("hello");
            }
        }
    });
</script>
</body>
</html>
```
## 双向绑定 需要使用 v-model

## 插槽slot以及自定义事件分发
```html
<!DOCTYPE html>
<html lang="en" xmlns:v-on="http://www.w3.org/1999/xhtml">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script src="./vue.min.js"></script>
</head>
<body>
<div id="app">
   <todo>
       <todo-title slot="todo-title" v-bind:title="title"></todo-title>
       <todo-lists slot="todo-lists" v-for="(list,index) in lists" v-bind:list="list"
                   v-bind:aria-colindex="index" v-on:remove="removelist(index)"></todo-lists>
   </todo>
</div>


<script>
    Vue.component("todo",{
        template:'<div>' +
            '<slot name="todo-title"></slot>' +
            '<ul>' +
            '<slot name="todo-lists"></slot>' +
            '</ul>' +
            '</div>'
    });
    Vue.component("todo-title",{
        props:['title'],
        template: '<div>{{title}}</div> '
    });
    Vue.component("todo-lists",{
        props:['list','index'],
        template: '<li>{{list}} <button v-on:click="remove">删除</button></li> ',
        methods:{
            remove:function (){
                //自定义事件分发
                this.$emit('remove');
            }
        }
    });


    var vm = new Vue({
        el: "#app",
        data: {
            title:"这是一首简单的小情歌",
            lists:["十年","十一年","我相信"],
            message: "hello!"
        },
        methods:{
            removelist: function (index){
                this.lists.splice(index,1);
            }
        }
    });
</script>
</body>
</html>
```
splice方法
splice(位置，删除个数，新增数据，新增数据);
